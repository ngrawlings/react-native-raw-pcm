# react-native-raw-pcm

Allows recording and playback of pcm buffers

## Installation

```sh
npm install react-native-raw-pcm
```

## Usage

```js
import RawPcm from "react-native-raw-pcm";

// ...

const result = await RawPcm.multiply(3, 7);
```
